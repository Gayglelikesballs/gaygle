# 💅 Contributing to Gaygle

First off: thank you for wanting to make the internet gayer. We appreciate you.

## How to Contribute

### 1. Make it Gayer
The #1 priority of any contribution is increased gayness. If your PR doesn't make the site at least 10% gayer, it will be rejected.

### 2. Add More Rainbows
We currently have approximately 847 rainbow gradients. This is not enough. There is no upper limit to rainbows.

### 3. Write Funnier Roasts
Search results need to be funny. Not "haha" funny. "I need to screenshot this and send it to my group chat" funny.

### 4. Break SafeSearch Even More
SafeSearch is already permanently off, but we believe it can be MORE off. Find a way.

## Code Style

- Tabs vs Spaces: We use whatever pisses off more people
- Comments: Should be sassy
- Variable names: Should be fabulous
- Error handling: `catch (e) { console.log("slay anyway 💅"); }`

## Pull Request Process

1. Fork the repo
2. Make your changes
3. Open a PR with a gay title
4. CRAWLER-RANK will judge your code
5. If CRAWLER-RANK says "slay" → merged
6. If CRAWLER-RANK says "mid" → rejected
